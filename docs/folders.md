# Folder Structure Documentation

This document explains the root-level directories and their purposes within the Confidential Donation Platform repository.

```text
Team1-SpeedRun/
├── backend/          # Optional Node.js & Express server for off-chain caching and API indexing.
│   ├── data/         # Mock local database (JSON format) for development.
│   ├── server.js     # Express server entry point.
│   └── package.json  # Backend dependencies (express, cors, dotenv).
│
├── contracts/        # Solidity smart contracts.
│   └── ConfidentialDonation.sol # Core logic for campaigns and eERC payload storage.
│
├── docs/             # Technical documentation.
│   ├── api.md        # API definitions for smart contracts and backend.
│   ├── architecture.md # System architecture overview.
│   └── folders.md    # This file.
│
├── frontend/         # React and Vite Web3 User Interface.
│   ├── src/
│   │   ├── abi/          # Compiled ABI JSON files for smart contract interaction.
│   │   ├── components/   # Reusable UI components (Dashboard, ConnectWallet, Toast).
│   │   ├── utils/        # Ethers.js integration and eERC mock service layer.
│   │   ├── App.jsx       # Main application routing and shell.
│   │   └── index.css     # Global theme and reset styles.
│   └── .env              # Auto-generated file storing the deployed contract address.
│
├── scripts/          # Hardhat deployment and operational scripts.
│   └── deploy.js     # Deploys the contract to Avalanche Fuji and exports addresses.
│
└── artifacts/        # Auto-generated by Hardhat after compilation. Contains contract ABIs.
```
